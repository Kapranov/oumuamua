# Create Cryptocurrency Trading Bot in Elixir

Repository to make following "Create Cryptocurrency Trading Bot in
Elixir" easier.

## An initial project

```
iex> mix new ./ --app oumuamua --umbrella
```

## Running

```
iex> -S mix
```
